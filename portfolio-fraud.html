TalkingData Ad Fraud Detection

Project information

  • Category: Machine Learning
  • Client: Study
  • Project date: 30 Set, 2021
  • Project URL: https://github.com/natmurad/TalkingDataAdtrackingFraudDetection

Script to predict fraud clicks in ads. Working with this kind of dataset is hard because it is a very large dataset and the classes are very unbalanced. Here, we have 2 classes, 0 and 1, indicating id the click resulted in a download of the app or not. The script process the file in small chunks and apply a filter creating a new file with a similar number of observations in both classes. Then, this balanced file is read and it is made an exploraty data analysis and feature engineering in order to get a better fit of the model. Once it is done, it is trained a XGBoost and a LightGBM algorithms to predict if the download was executed or not.
